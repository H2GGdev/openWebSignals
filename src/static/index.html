OpenWebSignals API

by h2gg

Overview

OpenWebSignals is a web API providing tools for DNS lookup, SSL information, and web analysis. Below are the available endpoints and their usage.

GET /health

Check API health status.

Response: 
{ "status": "ok" }

GET /analyze

Analyze a website for signals and metrics.

Request Body: 
{ "url": "https://example.com" }
Response: 
{ "analysis": { ... } }
